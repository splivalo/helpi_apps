# Helpi App - Unified Customer & Student Flutter App

Unified mobile Flutter app for both Customer (Senior) and Student roles.

## Tech Stack

- Flutter 3.x
- Dio 5.x + JWT token storage
- Riverpod (`flutter_riverpod ^2.6.1`)
- SignalR (`signalr_netcore ^1.4.4`)
- Localization with `AppStrings` (HR/EN)

## Quick Start

```bash
flutter pub get
flutter run
```
